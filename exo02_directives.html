A directive is an HTML attribute wich begin by "data-ng-" or "ng-".
"data-ng-" is HTML5 valid.

General

data-ng-app

/!\ DO NOT COPY THIS! WRONG LETTERS!!! /!\
It allows to declare an AngularJS application.
Here the name of the application is "my_app", but it also can be empty.
If the application has a name, you HAVE TO initialize it in a "script" tag.
/!\ AngularJS does not act outside application /!\
/!\ Only ONE application BLOCK per page! Even calling two times the SAME application WILL NOT works. /!\

data-ng-init

It allows to declare an AngularJS variable.
Here the variable name is "nbr" and its value is the integer "25".
Here the string "str" contains "Hello" (declared between simple quotes).
Here the array "array" contains "[ 1, 2, 'Hello', 'World' ]" (declared between brackets).
Here is an object (declared between simple curly-braces).
Here is an array of objects. And so on...
You can also use this directive to declare multiple variables.
Spaces are not mandatory.

Data Binding

data-ng-model

It allow to use the value of an AngularJS variable for the tag with value.

data-ng-bind

It allow to use the value of an AngularJS variable for the tag with value.
And if you write something inside the input, the data will be bind inside the tag.
