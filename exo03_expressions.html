An expression is written between double curly-brace: {{}}
It allows to read variables, do maths operations and string concatenation.
Just follow JavaScript rules.
  • Read variable:
    • Number: {{nbr}}
    • String: {{str}}
    • Array: {{array}}
    • Object: {{obj}}
    • ...
  • Maths operations:
    • 10 + 2 = {{10 + 2}}
    • 10 + 2 = {{10 - 2}}
    • 10 * 2 = {{10 * 2}}
    • 10 / 2 = {{10 / 2}}
    • 10 % 2 = {{10 % 2}}
    • (9 - 5) * 3 + 2 = {{(9 - 5) * 3 + 2}}
    • nbr1 + nbr2 = {{nbr1 + nbr2}}
    • ...
  • Concatenation:
    • {{"Hello" + " World"}}
    • {{str1 + " " + str2}}
    • {{str1 + " " + nbr}}
